# Common Error Messages

A reference of error messages you may encounter and what they mean.

| Error | Likely cause | Fix |
|---|---|---|
| "Permission denied" | Your role does not include this action. | Ask your account admin for the required role. |
| "Scan failed" | Scanner could not process the input file. | Check the file format and try re-uploading. |
| "Integration auth failed" | API key or credentials expired. | Regenerate credentials in Administration → Integrations. |
| "License not found" | Unknown SPDX identifier. | Check the license name against the SPDX license list. |
