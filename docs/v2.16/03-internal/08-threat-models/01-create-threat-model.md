# Create a Threat Model

> [!NOTE]
> **Required license feature:** `threat`

1. Navigate to **Internal → Threat Models**.
2. Click **Create**.
3. Select the **project** this threat model applies to.
4. Enter a **name** for the threat model.
5. Click **Save**.

The threat model is created and you can start editing it in the [detail view](02-threat-detail-stride-linddun.md).
