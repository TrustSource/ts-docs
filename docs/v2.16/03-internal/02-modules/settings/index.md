# Module Settings

Eighteen tabs of module-level configuration. Tab visibility depends on your role and on the resource-based field-level permissions; some tabs override project-level defaults.

## In this chapter

| Page | Status | Scope |
|---|---|---|
| [General](01-general.md) | _Scaffold_ | Name, version, description, tags, project assignment, related users. |
| [Legal](02-legal.md) | _Scaffold_ | Module-level legal settings; can override project; wizard, import. |
| [Obligations](03-obligations.md) | _Scaffold_ | License obligations at module level; propagate. |
| [Viability](04-viability.md) | _Scaffold_ | EOL / activity / community thresholds. |
| [OSSF](05-ossf.md) | _Scaffold_ | OpenSSF Scorecard thresholds. |
| [Versioning](06-versioning.md) | _Scaffold_ | Version drift thresholds and jump mode. |
| [Module Classification (MDR / IVDR)](07-module-classification-mdr-ivdr.md) | _Scaffold_ | Medical-device classification flags. |
| [CIA](08-cia.md) | _Scaffold_ | Module-level CIA requirements. |
| [Capabilities](09-capabilities.md) | _Scaffold_ | Feature / capability assertions per module. |
| [Encryption](10-encryption.md) | _Scaffold_ | Module-level crypto requirements. |
| [End of Life](11-end-of-life.md) | _Scaffold_ | Module-level EOL policy. |
| [Quality](12-quality.md) | _Scaffold_ | Module-level quality settings. |
| [Releases](13-releases.md) | _Scaffold_ | Per-module release configuration; analysis / scans visibility. |
| [Whitelists & Blacklists](14-whitelists-blacklists.md) | _Scaffold_ | Per-module W/B lists; can override project. |
| [CVE Whitelist](15-cve-whitelist.md) | _Scaffold_ | Suppress specific CVEs at module level. |
| [Custom Components](16-custom-components.md) | _Scaffold_ | Manual entries for in-house libraries that no scanner finds. |
| [SPDX Export](17-spdx-export.md) | _Scaffold_ | SPDX UUID and export configuration. |
| [Danger Zone](18-danger-zone.md) | _Scaffold_ | Archive, delete module, delete history. |
