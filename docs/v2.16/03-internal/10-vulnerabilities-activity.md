# Vulnerabilities Activity Feed

> [!NOTE]
> **Required role:** `company_security_manager`

The Vulnerability Activity feed provides a chronological view of all vulnerability events across your company — new CVEs discovered, status changes, muting decisions and resolution actions.

## What it shows

- **New vulnerabilities** — CVEs newly matched to components in your modules.
- **Status changes** — when a vulnerability's status changes (e.g. from "open" to "mitigated").
- **Muting actions** — when a vulnerability is muted (accepted as known risk).
- **Resolution actions** — when a vulnerability is resolved (component updated, workaround applied).

Filter by project, module, severity and date range.
