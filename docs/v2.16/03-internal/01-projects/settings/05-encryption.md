# Encryption (Project)

> [!NOTE]
> **Required role:** users with write permission on project encryption settings

The Encryption tab manages cryptographic algorithm requirements and export control settings at the project level.

## What it does

- Define which **cryptographic algorithms** are allowed or required for this project.
- Configure **export control** classifications that affect how components with crypto capabilities are evaluated.
