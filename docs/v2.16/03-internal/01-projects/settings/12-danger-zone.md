# Danger Zone (Project)

> [!NOTE]
> **Required role:** users with `project.danger.delete` execute permission

The Danger Zone tab contains destructive operations for this project.

## Delete project

Permanently deletes the project and optionally all its modules.

- If the project **has modules assigned**, you must either unassign them first or choose to force-delete all modules along with the project.
- If the project **has no modules**, the delete button is enabled directly.

> [!CAUTION]
> Deleting a project is **permanent and cannot be undone**. All project settings, history and — if force-deleted — all module data, scans and approvals are lost. Consider archiving instead if you may need the data later.
