# CSAF Documents

> [!NOTE]
> **Required role:** `compliance_manager`, `manager` or `account_admin`
> **Required license feature:** `csaf`

CSAF (Common Security Advisory Framework) documents are structured security advisories that describe vulnerabilities in your products and the remediation steps you have taken.

## What you can do

- **Create** CSAF advisories with structured metadata.
- **Edit** across multiple tabs: Metadata, Affected Products, Remediation, History.
- **Export** as CSAF JSON for publication.
- **Bulk-create** from SBOM — generate advisories for all known vulnerabilities in a module.
