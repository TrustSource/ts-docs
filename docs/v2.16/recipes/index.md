# How-To Recipes

The chapters in this manual are organized like the app's left navigation — useful when you know **where** to click. This page is the other navigation: **task-oriented recipes** for users who know **what they want to achieve** but not yet which menu it lives in.

Each recipe walks an end-to-end goal from start to finish, crossing whatever menu boundaries it needs to. They link back into the reference chapters for the details.
