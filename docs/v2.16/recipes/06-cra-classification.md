# Recipe: CRA Product Classification

Classify a product under the EU Cyber Resilience Act.

## Steps

1. Create a [product](../03-internal/03-products/index.md) (requires `products` license).
2. Open the [Classification tab](../03-internal/03-products/04-tab-classification-cra.md).
3. Select the CRA tier: STANDARD, IMPORTANT_1, IMPORTANT_2, or CRITICAL.
4. Fill in the [Contacts tab](../03-internal/03-products/07-tab-contacts.md) — manufacturer info is required.
5. Document [Misuse Cases](../03-internal/03-products/09-tab-misuse.md).
6. [Link approved releases](../03-internal/03-products/10-link-with-release.md) as solutions.

See [Background: CRA](../../shared/background/01-cra-cyber-resilience-act.md) for regulatory context.
