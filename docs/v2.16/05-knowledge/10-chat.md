# AI Chat Assistant

> [!NOTE]
> **Required license feature:** `chat`
> **Status:** BETA

The AI Chat Assistant is powered by AWS Bedrock and provides conversational help with compliance questions, component lookups and general ***TrustSource*** usage.

## Two modes

| Mode | Where | How to open |
|---|---|---|
| **Panel** | Right side, 380px wide (resizable to 760px) | Click the chat icon in the right bar |
| **Full page** | Dedicated route at `/chat` | Navigate via menu or URL |

## Features

- Session-based conversations with history.
- Vote (thumbs up/down) and copy message.
- Clear history and reset session.
