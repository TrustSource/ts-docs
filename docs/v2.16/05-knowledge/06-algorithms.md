# Algorithms

The Algorithms database catalogs cryptographic algorithms with their FIPS status, deprecation timeline, key lengths and post-quantum readiness.

## What you can do

- **Search** by algorithm name or standard.
- **View FIPS status** — approved, deprecated, or retired.
- **Check key lengths** — minimum recommended sizes.
- **Track PQC readiness** — post-quantum migration status.
