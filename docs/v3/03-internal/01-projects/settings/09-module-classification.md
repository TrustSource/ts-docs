# Module Classification (Project, MDR/IVDR)

> [!NOTE]
> **Required role:** users with `project.m-classification.settings` write permission
> **Required account type:** Medical

The Module Classification tab lets you set MDR (Medical Device Regulation) and IVDR (In Vitro Diagnostic Regulation) classifications at the project level.

## Fields

| Field | Description |
|---|---|
| **MDR Classification** | Medical Device Regulation classification for this project. |
| **IVDR Classification** | In Vitro Diagnostic Regulation classification. |

## Actions

- **Import from other project** — copy classification from another project.
- **Propagate** — push classification to all modules and other projects.

> [!NOTE]
> This tab is only visible for Medical accounts. See [Account Types](../../../00-getting-started/02-account-types.md) for edition details.
