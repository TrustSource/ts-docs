# Module Import (Project)

> [!NOTE]
> **Required role:** users with `project.ModuleImportTab.import` execute permission

The Module Import tab lets you bulk-import modules into this project from files or SPDX documents.

## How to use it

1. Click **Import**.
2. Select a file (SPDX, JSON or CSV format).
3. Review the import preview — modules to be created and components to be added.
4. Confirm the import.

The import history table shows all past imports with their status.
