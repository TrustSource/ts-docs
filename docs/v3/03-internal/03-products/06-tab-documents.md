# Tab: Documents

> [!NOTE]
> **Required license feature:** `products`

Structured document storage with predefined categories: user manual, secure setup guide, product description, SBOM and CEDOC (Cyber-security European Declaration of Conformity). One file per category, with URL and upload metadata.
