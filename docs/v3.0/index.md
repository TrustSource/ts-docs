# TrustSource v3.0 (Community Edition)

> [!NOTE]
> The Community Edition documentation is **coming soon**. The v3.0 release is in preparation.

***TrustSource*** Community Edition is a free, self-hosted, open-source edition of the TrustSource platform. It provides a subset of features available in the commercial editions.

## What to expect

- Self-hosted deployment via Docker Compose.
- Core compliance features: projects, modules, scans, basic reporting.
- Open-source under a permissive license.

## Differences from v2.16

The Community Edition does not include enterprise features such as:

- Multi-company hierarchy
- Keycloak/SSO integration
- Products module (CRA classification)
- Threat modelling
- AI Chat Assistant
- SOUP reports (medical)
