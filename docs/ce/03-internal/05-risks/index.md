# Risks

> [!NOTE]
> **Required role:** `developer`, `compliance_manager`, `manager` or `account_admin`
> **Required license feature:** `risks`

The Risks module provides a **risk register** with financial metrics, portfolio views and task management. It lets you document, assess and track risks across your software projects.
