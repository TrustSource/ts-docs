# Support & Contact

## Getting help

- **This documentation** — search or browse for answers.
- **In-app help** — click the help icon in the right bar.
- **AI Chat** — use the AI assistant (requires `chat` license feature).

## Contact support

**Email:** [support@trustsource.io](mailto:support@trustsource.io)

When opening a ticket, include:

1. Your company name and account type.
2. The page or feature where the issue occurs.
3. Steps to reproduce.
4. Error messages (screenshot or text).
5. Browser and OS (for UI issues).
